# Number Guessing Game

## Description
This is a Java console-based Number Guessing Game developed as part of the Oasis Infobyte Internship.

## Features
- Three difficulty levels
- Random number generation
- Limited attempts
- Scoreboard
- Multiple rounds
- Play Again option

## Technologies Used
- Java
- ArrayList
- Random
- Scanner

## How to Run

1. Compile
```
javac NumberGuessingGame.java
```

2. Run
```
java NumberGuessingGame
```
